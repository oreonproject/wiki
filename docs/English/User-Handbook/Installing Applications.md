# Installing Applications on Oreon

Oreon uses GNOME Software for installing apps.

![GNOME Software Center in Oreon](/img/softwarecenter.png)

## Using GNOME Software

1. Open **Software** from the Activities overview.
2. Search for the application.
3. Click **Install**.

![GNOME Software Center in Oreon](/img/softwarecenter1.png)

## Using the Terminal

```bash
sudo dnf install <package-name>
