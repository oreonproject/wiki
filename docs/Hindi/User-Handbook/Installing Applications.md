# ओरेऑन पर एप्लिकेशन इंस्टॉल करना

ओरेऑन ऐप्स इंस्टॉल करने के लिए ग्नोम सॉफ़्टवेयर का उपयोग करता है।

![ओरेऑन में ग्नोम सॉफ़्टवेयर सेंटर](/img/softwarecenter.png)

## ग्नोम सॉफ़्टवेयर का उपयोग करना

1. एक्टिविटीज़ ओवरव्यू से **Software** खोलें।
2. एप्लिकेशन खोजें।
3. **Install** पर क्लिक करें।

![ओरियन में ग्नोम सॉफ़्टवेयर सेंटर](/img/softwarecenter1.png)

## टर्मिनल का उपयोग करना

```bash
sudo dnf install <package-name>
```
