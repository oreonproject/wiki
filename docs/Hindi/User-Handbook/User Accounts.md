# उपयोगकर्ता खातों का प्रबंधन

**सेटिंग्स → सिस्टम → उपयोगकर्ता** पर जाएँ।

![उपयोगकर्ता पैनल](/img/users-panel.png)

## एक उपयोगकर्ता जोड़ना

**उपयोगकर्ता जोड़ें** पर क्लिक करें और निर्देशों का पालन करें।

![उपयोगकर्ता जोड़ें संवाद](/img/add-user.png)
