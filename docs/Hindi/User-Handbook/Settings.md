# सिस्टम सेटिंग्स

सिस्टम मेनू से **सेटिंग्स** खोलें।

![सेटिंग्स अवलोकन](/img/settings-overview.png)

## नेटवर्क

**नेटवर्क** के अंतर्गत वाई-फ़ाई और ईथरनेट कनेक्शनों का प्रबंधन करें।

![नेटवर्क पैनल](/img/network-panel.png)

## डिस्प्ले

**डिस्प्ले** के अंतर्गत रिज़ॉल्यूशन, स्केलिंग और मॉनिटर लेआउट समायोजित करें।

![डिस्प्ले पैनल](/img/display-panel.png)
