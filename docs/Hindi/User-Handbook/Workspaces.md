# वर्कस्पेस और मल्टीटास्किंग

GNOME डायनामिक वर्कस्पेस का उपयोग करता है।

![वर्कस्पेस अवलोकन](/img/workspaces-overview.png)

## एक वर्कस्पेस बनाना

किसी विंडो को खाली वर्कस्पेस पूर्वावलोकन पर खींचें या सेटिंग्स में एक कस्टम वर्कस्पेस काउंट सेट करें।

![वर्कस्पेस निर्माण](/img/workspace-creation.png)

## अन्य विंडो कीबोर्ड शॉर्टकट

- सुपर + पेज अप/पेज डाउन
- सुपर + टैब
- सुपर + एरो कीज़
