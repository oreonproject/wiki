# आवश्यक कीबोर्ड शॉर्टकट

| शॉर्टकट | क्रिया |
|----------|--------|
| सुपर (विंडोज़ की) | ऐप मेनू खोलें |
| सुपर (विंडोज़ की) + A | एप्लिकेशन स्क्रीन दिखाएँ |
| सुपर (विंडोज़ की) + L | स्क्रीन लॉक करें |
| Alt + F4 | विंडो बंद करें |
| प्रिंट स्क्रीन | स्क्रीनशॉट लें |
