# Atajos de teclado esenciales

| Atajo | Acción |
|----------|--------|
| Super (tecla Windows) | Abrir menú de aplicaciones |
| Super (tecla Windows) + A | Mostrar pantalla de aplicaciones |
| Super (tecla Windows) + L | Bloquear pantalla |
| Alt + F4 | Cerrar ventana |
| Imprimir pantalla | Hacer captura de pantalla |
