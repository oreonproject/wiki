# Instalación de aplicaciones en Oreon

Oreon utiliza GNOME Software para instalar aplicaciones.

![Centro de software GNOME en Oreon](/img/softwarecenter.png)

## Uso de GNOME Software

1. Abra **Software** desde la vista general de Actividades.
2. Busque la aplicación.
3. Haga clic en **Instalar**.

![Centro de software GNOME en Oreon](/img/softwarecenter1.png)

## Uso del terminal

```bash
sudo dnf install <nombre-del-paquete>
