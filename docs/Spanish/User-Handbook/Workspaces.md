# Espacios de trabajo y multitarea

GNOME utiliza espacios de trabajo dinámicos.

![Descripción general del espacio de trabajo](/img/workspaces-overview.png)

## Creación de un espacio de trabajo

Arrastre una ventana a la vista previa del espacio de trabajo vacío o establezca un número de espacios de trabajo personalizado en la configuración.

![Creación de un espacio de trabajo](/img/workspace-creation.png)

## Otros atajos de teclado para ventanas

- Super + Av Pág/Re Pág
- Super + Tab
- Super + Teclas de flecha
