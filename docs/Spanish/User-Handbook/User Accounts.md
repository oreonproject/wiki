# Administración de cuentas de usuario

Vaya a **Configuración → Sistema → Usuarios**.

![Panel de usuarios](/img/users-panel.png)

## Añadir un usuario

Haga clic en **Añadir usuario** y siga las instrucciones.

![Cuadro de diálogo Añadir usuario](/img/add-user.png)
